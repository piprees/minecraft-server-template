# Product

> **This is an example product doc.** It describes the design thinking behind the web surfaces that ship with this template. Adapt the brand personality, users, and references to fit your own server.

## Register

product

## Users

A small group of friends, invited via Discord, most non-technical. They land on these pages exactly once (to download the modpack and join) or occasionally (to check status, browse the map). Context: excited to join a friend's server, or checking if it's up before launching. The game is the destination; the website is the trailhead.

## Product purpose

The server's web surfaces (pack page, map, status, mods) exist to get friends into the game with zero friction and to provide at-a-glance server health. Success: someone downloads the pack, joins the server, and never thinks about the website again - but remembers it felt considered, not like a random Minecraft server page.

## Brand personality

Rugged, mythic, wild.

The server runs Terralith, Incendium, and Nullscape - terrain mods that make the world feel ancient, vast, and unexplored. The tone is a group of friends heading into unmarked territory together, not a public server recruiting strangers. Confident and understated, never loud or performative.

## Anti-references

- **Generic MC server lists** - MCSL/Planet Minecraft neon-green Minecraft-font "JOIN NOW" energy. This is a private server; it doesn't recruit.
- **Corporate SaaS dashboards** - Vercel/Linear dark-mode slickness. Could be anything, belongs to no one.
- **Overstimulating gaming sites** - Hypixel/CurseForge busy-loud-ad-laden scale-optimised design. This is for a small group, not thousands.
- **Twee/cutesy indie games** - Pastel, rounded, soft. Too gentle for an adventure server with Incendium's Nether.

## Design principles

1. **The landscape does the talking** - Terralith and Incendium make the world feel ancient and vast. Let that carry the identity. The chrome gets out of the way; the world is the hero.
2. **Quiet authority** - Like an Ordnance Survey map: precise, competent, no need to shout. Every element earns its place.
3. **Discovery over instruction** - The Tunic/Monument Valley feel: reward curiosity, don't over-explain. Progressive disclosure; details unfold when sought.
4. **Utility with soul** - Product register, but character lives in the materials (colour, texture, type weight), not in decoration. Welsh slate, not Welsh dragons.
5. **Private, not performative** - This is a trailhead for friends, not a storefront for strangers. Intimacy over scale.

## References

- **Visit Wales / Welsh heritage** - Slate, moss, weathered stone. The Welsh landscape as design language - dark, textured, mineral.
- **Ordnance Survey / cartographic** - Contour lines, grid references, the quiet functional beauty of a well-made map. Authority through precision.
- **Tunic / Monument Valley** - Isometric, geometric, mysterious. Discovery-oriented. Rewarding without hand-holding.

## Accessibility & inclusion

WCAG 2.2 AA. Standard contrast ratios, keyboard navigation, screen reader basics. No known specific accommodations needed - the audience is 16+ friends on desktop (Minecraft requires it). Mobile is secondary but should work for checking status/map on phones.
