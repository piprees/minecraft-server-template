# Brand assets

Place your server's brand assets here. They are used by the modpack builder
for the download page and by nav-proxy for the site navigation bar.

## Expected files

| File | Format | Size | Used by |
|------|--------|------|---------|
| `icon.svg` | SVG | any | Nav bar, favicon, Discord embed |
| `logo.svg` | SVG | any | Download page header |
| `cover.png` | PNG | 1920x1080 recommended | Download page hero / social preview |
| `favicon.ico` | ICO | 32x32 / 64x64 | Browser tab icon |

All files are optional. The platform ships placeholder assets that are used
when no override is provided.
